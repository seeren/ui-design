# ui-design
**Override ui design for JavaScript ui components**
Compatible with bootstrap class, this package override some component design to provide more material css

## Features
* `ui`
* `ui btn btn-{level}`
* `ui card card-{n-depth}`
* `ui cross`


## Installation
Install with [npm](https://www.npmjs.com/package/ui-design)
```
npm install ui-design
```
